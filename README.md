# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록

- [ ] 숫자 생성/검증
- [ ] 랜덤 숫자 생성/검증
- [ ] 컴퓨터와 사용자의 숫자 비교
- [ ] 게임 입/출력 기능 구현
- [ ] 게임 진행에 대한 컨트롤러 기능 구현
    - [ ] 게임 시작/진행/종료 구현
    - [ ] 3스트라이크시 게임 진행 여부 판별 및 재시작/종료 구현
